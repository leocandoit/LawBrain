> 回调函数
>
> * **实时流式输出**
>   回调函数可以在模型生成答案时实时接收和处理输出，从而实现流式响应，让用户可以更快地看到部分结果，而不必等到整个生成过程完成。
> * **日志记录和调试**
>   回调可以记录每个步骤的状态、执行进度以及潜在的错误信息，有助于开发者在调试过程中了解每个环节的具体表现，快速定位问题所在。
> * **异步事件处理**
>   在异步或并行处理环境下，回调函数能够在某个操作完成后及时触发后续动作，使得整个链式流程更高效地运行，而无需阻塞等待每个步骤的完成。
